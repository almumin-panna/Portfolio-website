## 🛠️ Technology Stack
- **[React.js](https://react.dev/)** - UI Component Library
- **[Tailwind CSS v4](https://tailwindcss.com/)** - Premium glassmorphism design and responsive layout
- **[Framer Motion](https://www.framer.com/motion/)** - Smooth scroll reveal animations and dynamic effects
- **[Vite](https://vitejs.dev/)** - Lightning-fast build tool
- **[Lucide React](https://lucide.dev/)** - Beautiful and consistent iconography

## ✨ Key Features
- 🎨 **Premium Dark Mode UI**: A highly modern, visually stunning aesthetic blending dark themes with vibrant accent colors (Electric Blue & Neon Cyan).
- 🪟 **Glassmorphism Design**: Sleek translucent cards for a state-of-the-art tech feel.
- 🚀 **Interactive Sections**:
  - **Hero**: Dynamic typing effect with rotating professional titles and an animated gradient border ring.
  - **About**: Photo gallery carousel with crossfade transitions showcasing lifestyle and professional shots.
  - **Skills Toggle**: A dual-track interactive toggle switching seamlessly between "Civil Engineering" and "Web Development" skill sets.
  - **Experience & Education**: Beautiful timeline-based professional history.
- 📱 **Floating Social Menu**: Fixed social contact button expanding on click to display WhatsApp, Facebook, YouTube, and Instagram links.
- ⚡ **Responsive Layout**: Perfectly optimized for Mobile, Tablet, and Desktop screens.

## 🏗️ Design Concept
The design language reflects a perfect bridge between a professional **Civil Engineer** and a high-tech **Web Developer/WordPress Expert**. It features subtle geometric background patterns reminiscent of architectural blueprints, balanced with the fluid, glowing elements of modern web applications.

## 💻 Running the Project Locally

To get a local copy up and running, follow these simple steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/almumin-panna/Portfolio-website.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd Portfolio-website
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:5173/`
